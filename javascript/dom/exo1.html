Aprendendo a usar DOM em javascript

Como podemos acessas os elementos html direto no javascrtip

  1. por Marca
  2. por Id
  3. por Name
  4. por Classe
  5. por Seletor

Por Marca quando chamamos o elemento pela tag ex: div, h1, img, ul, ol... getElementsByTagName('div').

Por Id quando chamamos pelo id que damos nas tags, lembrando que cada tag tem seu id(unido) getElementById().

por Class quando chamamos pelo classe das tags, lembando que varias tags podem ter o mesmo nome de classe getElementsByClassName() .

por Seletor quando chamamos document.querySelector('div#nomedoid'). document.querySelectorAll() esse all() é o plural.

Quando for por marca e tem mais de uma gat podemos definir qual será modificada usando uma numerção ou indice assim como no array ex: se tiver mais de uma tag p, e eu quero usar só a primeira fazemos assim getElementsByTagName('p')[0]. Lembrando que esse final zero é equivalente a primeira tag p do meu body.

Lembrando que quando vamos usar tags com o mesmo nome, temos que colocar getElementsBy. O getelements(s) tem que vim no plutal.

inner.Text = mostra a tag sem formatação ex tenh um strong. Já a innerHTML mostra tudo.
