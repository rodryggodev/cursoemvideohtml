Number e String Js

Lembrando que para pegar um entrada de dados no js, usamos o seguinte comando.

script let nome = window.prompt('qual o seu nome?'); console.log('bem vindo ' + nome); ou window.alert('bem vindo ' + nome). que irá mostrar um alerta na tela do navegador. script

Outra coisa muito importante na hora de fazer calculo em javascript, é que temos que formatar os valores, já que se isso não acontecer os valores passados ainda que sejam number(numero). Caso não tenha essa formatação irá ser lido como uma string e não era fazer a soma e sim uma contatenação, e o valor esperado não será somado e sim irão se juntar.

Ex: 10+10 não irá ser igual a 20, mas sim irá mostrar 1010 para que isso não ocorra vamos usar o format.Para os numeros inteiros iremos usar o Number.parseInt() e Number.parseFloat() para numeros reais. Mas essas duas opções servem so mente para valores inteiros e real, caso eu queria misturar os dois valores isso não irá funcionar.

Com a chegada do ecma6 tem a função Number() que é a mais usada, e ela tem a função de saber automaticamento se os valores passados são reais ou inteiros.

Interpolação

Interpolação são maneiras de como "concatenar variavel a uma saida de dados(output), em um alert,console.log".

let s='nome';

  • 'eu estou aprendendo s'// não faz interpolação, aqui vai sair o esse e nao o valor da variavel.
  • 'eu estou aprendendo' + s // concatenação e vai mostrar o valor da variavel corretamente
  • `eu estou aprendendo ${s}` // faz interpolação e a maneira atualizada de fazer template string
  • s.length // quantos caracteres a string tem.
  • document.write(`seu nome ${variavel.length()}`);
  • s.toUpperCase() // tudo para "MAIÚSCULO"
  • document.write(`seu nome ${variavel.toUppercase()}`);
  • s.toLowerCase() // tudo para minusculo
  • document.write(`seu nome ${variavel.toLowerCase()}`);
