Link Externo

O link externo aponta para um site que não é nosso

Guithub

Lembrando que quando se coloca só o link sem o target="_blank", o site externo abri na mesma pagina do nosso site. podemos também colocar o rel="external", para indicar que vai para fora do meu site. O target="_blank" serve para abrir site externo em uma outra aba do navegador.

Link Interno

Link interno serve para levar o usario para outras areas do noss site sem ele ter que ir para outra aba do navegador, basta setar dentro da tag link o nome do arquivo, ou o nome da pasta e do arquivo que ta dentro da pasta e pronto. nesse caso chamamos de setar o diretorio
